# 🛡️ ShadowScan

ShadowScan is an AI-powered Shadow IT & Cloud Exposure Monitoring Platform built to help organizations detect unauthorized SaaS usage, analyze risks, and improve security visibility in real time.

Modern employees use tools like ChatGPT, Dropbox, Telegram, Notion, and WeTransfer daily — often without security approval.

ShadowScan helps organizations regain visibility and control.

---

# 🚀 Features

## 🔍 Real-Time SaaS Detection
Detects unauthorized SaaS tools directly from the browser using a Chrome MV3 Extension.

## 🧠 AI-Powered Risk Analysis
Analyzes detected domains and services using threat intelligence APIs.

## 📊 Threat Intelligence Integration
Integrated with:
- VirusTotal
- Shodan

## 🚦 Smart Risk Classification
Automatically categorizes services into:
- Approved ✅
- Monitor ⚠️
- High Risk 🚨
- Blocked ❌

## 📋 Compliance Mapping
Maps detected tools against:
- GDPR
- HIPAA
- PCI-DSS

## 🚨 Real-Time Alerts
Instant notifications when suspicious or high-risk activity is detected.

## 📄 Executive PDF Reports
Generate one-click reports for leadership and compliance teams.

## 🖥️ SOC-Style Dashboard
Modern real-time dashboard for monitoring organization-wide SaaS activity.

## 🏢 Multi-Tenant Architecture
Supports multiple organizations and environments.

## 🐳 Dockerized Deployment
Fully containerized setup for easy deployment.
